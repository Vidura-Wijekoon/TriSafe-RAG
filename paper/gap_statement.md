# Research Gap Statement — TriSafe-RAG

Existing work addresses language preference in multilingual RAG, fairness in
retrieval-augmented systems, language bias in multilingual IR, hallucination
in RAG, and abstention in language models as separate, independent problems.

No existing framework jointly optimizes evidence grounding, cross-lingual
fairness, and uncertainty-aware selective abstention in a single end-to-end
trustworthy multilingual system for public-service AI in emerging regions.

**TriSafe-RAG directly addresses this gap.**
